Etiquetas Semanticas Estructurales

Es una etiqueta contenedora que no tiene semantica
Header: cabecera de un sitio o una seccion
main: define la seccion principal del documento
footer: pie de pagina de un sitio web o de una seccion del sitio
nav: representa una navegacion o botonera
article: articulo, representa una seccion de autocontenido (que por si sola explica)
aside: representa un contenido complementario o secundario
section: representa una seccion de contenido generico
address: representa una direccion fisica o cualquier tipo de informacion de contacto

Etiquetas de bloque VS etiquetas de linea

La etiqueta de bloque por excelencia es la div, una etiqueta de bloque es aquella que genera saltos de linea con sus etiquetas hermanas
La etiqueta de linea por excelencia es la etiqueta span, una etiqueta de linea solo ocupa el espacio necesario que tiene su contenido Hola soy otra span 游댶
